## Currently Building — nexagg

A multi-service data aggregation platform, containerized end-to-end:

- **Docker Compose** stack: PostgreSQL, MongoDB, and app services
- Custom **Riot Games API** client (puuid → match history via Match-v5)
- Custom **Chess.com API** client (archive-based game retrieval)
- Kaggle League of Legends dataset loaded into a typed, constrained, indexed PostgreSQL schema

```text
nexagg/
├── docker-compose.yml
├── clients/
│   ├── riot_client.py
│   └── chess_client.py
├── db/
│   └── schema.sql
└── ...
```
